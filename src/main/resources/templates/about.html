📘 Book Management System

This is a full-stack CRUD web application developed using Spring Boot, Spring Data JPA, Thymeleaf, and MySQL. The system manages books and users with features like add, edit, delete, search, and dashboard summaries.

  • ✔️ Add/Edit/Delete/View Books
  • ✔️ Assign books to users
  • ✔️ Admin dashboard for summaries
  • ✔️ Search books by keywords
  • ✔️ Clean, modern UI with CSS

Created by Arghya Biswas. Built to showcase CRUD and backend skills using Java & Spring Boot.
